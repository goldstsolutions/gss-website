Do's & Don'ts

These rules protect brand consistency across every touchpoint. When in doubt, refer back to this section.

✓ Always Do
  • Use Plus Jakarta Sans 800 for all display text and headings
  • Keep table headers navy with gold text — always, no exceptions
  • Use the gold left-border style for H1 headings in documents
  • Include a branded cover page on every client-facing deliverable
  • Use info boxes (gold-dim bg + gold left border) for key problems
  • Use "Gold St Solutions · Client · Confidential" in every footer
  • Apply the grid texture on dark cover backgrounds for depth
  • Use #D4B96A as your single gold source of truth
✗ Never Do
  • Use gold as a large area background — it's an accent only
  • Mix in other font families (no Inter, Roboto, or system fonts)
  • Use pure black (#000) for text — use dark navy or void
  • Use white page backgrounds in dark-theme documents
  • Skip the cover page on any client deliverable
  • Use gold text on light or white backgrounds — contrast fails
  • Use more than 2–3 colors on any single page element
  • Alter the wordmark tracking, weight, or casing
